Home

I'm a full stack web developer who specializes in (and loves) Node.js, a server-side JavaScript platform. I often work on tools for the back end and front end, but I also create beautiful websites for all my clients, using modern Web technologies such as HTML5, CSS3, and JavaScript, as well as several tools to help accomplish these with even more flexibility. This website itself is created to take full advantage of HTML5, CSS3, and the latest JavaScript trends where applicable. If you're interested, please contact me here. (Please make sure to give me your email in that form as well.)

Another thing I enjoy to do is develop many useful JavaScript tools. These help me in creating Web sites and other tools, but I keep them open source so they can not only help me create great things, but also others who are interested as well.

I do occasionally maintain a low-traffic blog full of professional and not-so-professional whatevers that cross my head. It's mostly targeted towards those with a coding background, so it predictably has quite a bit of jargon within it.

My passions aren't limited to coding. I also enjoy playing piano (sorry, no recordings yet), listening to music, and even creating it myself. My own music spans quite a few genres, but even the music I listen to is a bit broad. Oh, and I like reading anything intellectually interesting like IFLScience or BuzzFeed (and occasionally, less intellectually satisfying ones, too).
